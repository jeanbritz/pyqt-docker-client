# Docker Client GUI
Docker Client GUI written in Python

# Requirements
- PyQt 5.11 (https://github.com/baoboa/pyqt5/)
- Docker Python Library (https://github.com/docker/docker-py)
- YAML Python Library (https://pyyaml.org/)
